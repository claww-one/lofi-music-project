# n8n 自動發文工作流 (Google Gemini 版)

### 1. 核心異動
- **引擎:** OpenAI -> Google Gemini (gemini-1.5-flash)
- **認證:** 需使用 Google AI Studio API Key。

### 2. 獲取 Gemini API Key
1. 訪問 [Google AI Studio](https://aistudio.google.com/)。
2. 點擊 "Get API key" 並建立新金鑰。
